BYTEBED

Bytebeat bytecode virtual machine written in C++ for M5Stack Cardputers. This is the WASM version suitable for testing and sharing patches online. Source, binaries and wasm published in Github.

Use Bank and Patch buttons to browse examples. Toggle between different visualizations: + and - adjusts refresh rate. INFIX / TAB key changes between infix and postfix notation. Toggle AUTO to evaluate with enter or automatically on change. See Project information and Cheatsheet and Github for more details.
